# 🧠 Handwritten Digit Recognition Web App

An AI-powered web application built using **Deep Learning (CNN)** and **Django** that recognizes handwritten digits from uploaded images.

---

# 🚀 Features

✅ Handwritten digit recognition (0–9)  
✅ Deep Learning model using CNN  
✅ Trained using MNIST dataset  
✅ Django web integration  
✅ Modern responsive UI  
✅ Image preview before prediction  
✅ Clear button for resetting input  
✅ Real-time prediction system  

---

# 🛠️ Technologies Used

- Python
- Django
- TensorFlow / Keras
- NumPy
- Pillow
- HTML
- CSS
- JavaScript
